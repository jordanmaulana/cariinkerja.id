About the Role:

As a Mobile Engineer (Flutter), you will be responsible for building, enhancing, and maintaining high-quality mobile applications that deliver seamless user experiences across Android and iOS platforms. You will work closely with cross-functional teams including Backend Engineers, UI/UX Designers, QA, and Product Managers to transform business requirements and product designs into scalable, performant, and maintainable mobile solutions.

Job Description:

  • Develop and maintain mobile applications using Flutter

  • Translate UI/UX designs into responsive and high-performance applications

  • Integrate RESTful APIs

  • Implement effective state management solutions

  • Perform debugging and optimize application performance

  • Write clean, maintainable, and testable code

  • Collaborate closely with Backend, UI/UX, QA, and Product teams

  • Manage build and release processes to Google Play Store and Apple App Store

  • Conduct code reviews and drive continuous improvement initiatives



Qualifications

  • Minimum 3+ years of experience as a Mobile Developer

  • Proven production experience using Flutter

  • Strong understanding of mobile application lifecycle

  • Solid knowledge of HTTP networking and JSON parsing

  • Familiar with Git and CI/CD workflows

  • Good understanding of Clean Architecture and/or MVVM pattern

  • Basic knowledge of mobile security best practices (secure storage, certificate pinning, etc.)
